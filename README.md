任何+86都没有对tg的短信验证码进行风控——关于Telegram无法收到验证码的问题

#### Telegram收不到验证码 原因

1. 第三方 app 导致收不到验证码

2. 网络问题：中国 +86 手机号注册 telegram 必须开启翻墙，翻墙开启香港、台湾节点，肯定能收到验证码，不建议使用免费的vpn，网络不稳定，被人滥用，现在 telegram 风控严，防止封号建议使用稳定的梯子翻墙。
